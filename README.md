# easy_tf_mlp_many_classes
Aplicação que permite testar diferentes topologias de modelos MLP e permite escolher entre Classificação multiclasse e Classificação binária, sendo ambas também possibilitadas por meio de aproximação do resultado de uma Regressão.

O dataset testado é o winequality, que foi pré-processado e utilizado pelos modelos de Classificação multiclasse Padrão e Classificação Multiclasse com aproximação de uma Regressão.
